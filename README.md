# Debby Touch Demo Website

Luxury outreach demo website concept for Debby Touch, a Dubai-based makeup artist and beauty brand.

## Overview

This static demo presents Debby Touch as a premium beauty experience for bridal makeup, soft glam, events, photoshoots, hairstyling, and complexion-aware artistry for Dubai's multicultural clientele.

## Files

- `index.html` - full landing page structure and SEO metadata
- `styles.css` - responsive luxury editorial design system
- `script.js` - header state, scroll reveal, and portfolio filtering

## Preview

Open `index.html` directly in a browser.

## Notes

This is an outreach/demo concept. Final imagery, WhatsApp number, pricing, portfolio work, and business details should be replaced with approved Debby Touch assets and content before launch.
